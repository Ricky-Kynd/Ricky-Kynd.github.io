Parentheses ( )

The use cases for parantheses, in the context of programming is regarded generally in controlling the order of operations as well as supplying parameters to functions, constructors or methods. For example, this is a use case when executing a function:

var result = doSomething()

Brackets [ ]

Brackets are quite similar to parantheses except it is mainly used in arrays to contain different values to which an array variable is passed. It is also is used in a term known as bracket notation where you would access properties or value key types if using objects, which is another use case. Here's a simple example:

var myArray = [ "string", 3, ]; get = object[property_name]; object[property_name] = set;

Braces { }

Braces are used to group statments and or declarations. These are commonly found in if statements, loops, methods and setting constructors. Here is a simple example:

var braces = myBraces(paranthesis) { //here's my curly braces doing its thing. };

Single quotes and Double quotes

Single quotes and double quotes are used for surrounding string literals. They both do the same job essentially but cannot be used more the once when defining a string. For instance, in this example you will see how it used by itself.

var areYouSingle = 'single quote coming through'; var makeItADouble = "double shot of whatever you have";

var seeingDoubles = "won't you pour me a drink already; var yesIAmSingle = 'I think the name of the bar is "Palisades"';
