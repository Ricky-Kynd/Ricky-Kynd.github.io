Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth

Control flow is quite simply the order of set conditions and statements which are then executed in steps usually written from top to bottom. A simple example I can think of from my frustrations this morning as I write this is the insertion of my USB flash drive. Firstly one would visually or mentally prepare to insert the drive into the free port, secondly, they would make their first attempt at inserting the drive the correct way, if that doesn't work, they would move on to the third step of flipping the flash drive the other way around and inserting it to achieve their goal. commented out in that function would be stated as "the number of attempts to insert this drive varies in terms of urgency - the faster you go the bigger the mess".

Explain the difference between accessing data from arrays and object literals

Here is a list of things you can do with an array. Nest one array with another array, access and modify array data with indexes, access multi-dimensional arryas with indexes and of course manipulate the arrays with the push, pop, shift and unshift methods. Objects on the other hand have properties and values which can be accessed with the dot operator as well as using bracket notation. You can also access object properties with variables update, add, and delete objects. These are the just the list of things you can do with arrays and objects. Explain what functions are and why they are useful

Functions are a basically a block of code (a set of methods) used or called upon to perform a certain task. The reason they are so useful is because they can be reused or refactored (probably using that word wrong) with different parameters to be used in different situations to produce the desired result.
